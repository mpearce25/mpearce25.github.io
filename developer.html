Virtual 3D Rubiks Cube

After becoming obsessed with Rubik's cubes, I attempted to code a working 3D model in Java. Powered by the JMonkey 3D engine, users can navigate around the cube and use keyboard shortcuts to control the movement of the cube. Additionally, with the help of my friend, I was able to implement a solving algorithim that can solve the cube from any state.

Hybrid IDE

Hybrid allows users with minimal knowledge of HTML and CSS to design fully functional websites by simply clicking on elements they want to appear on their webpage. This includes the ability to precisely control the style of images, text, and other elements. Additionally, the large text area allows users to maintain full control over their code and program in additional features that Hybrid does not yet support.

Color Sampler

Color Sampler was designed to simplify the difficult process of choosing colors when designing webpages and apps. By allowing users to sample colors from any content on their screen it is simple to quickly capture and copy colors. Additionally, the app allows users to create lists of favorites so they can easily acess previously used colors.

Jump

Designed for my AP Computer Science final, Jump was the first game I had ever built. The goal of the game is to stay alive for as long as possible by quickly jumping from platform to platform. As the users score gets higher, more and more platforms dissapear drastically increasing the difficults. The game is also very customizable allowing the user to change the background color, while the text and ball color randomly change.

Central Limit Theorem Proof

Created for my AP Stats class, this application allows the user to quickly simulate taking millions of samples from groups of varying sizes. While the statistics in the bottom of the app provide numerical evidence of the Central Limit Theorem, the user also has acess to the sample result from each group. By graphing this data the user can create visuals demonstrating this theorem.
